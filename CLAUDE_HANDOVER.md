# Project Handover: SQD-APP (Aircraft Maintenance QA System)

This document provides a comprehensive state-of-the-union for the **SQD-APP**, a full-stack Quality Assurance platform for aviation maintenance. Use this as context for Claude Code or any subsequent AI agent.

## 1. Project Objective
Build a robust, role-based system for aircraft maintenance surveillance.
- **Key Features:** Dynamic branching templates, task assignment, finding/audit trail management, and role-based access control (RBAC).
- **Core Workflow:** Admin creates Template -> Manager assigns Task -> Staff executes Form -> Findings are raised -> Follow-up actions are tracked.

## 2. Tech Stack
- **Backend:** Node.js (v20+), Express, TypeScript, Prisma ORM, PostgreSQL.
- **Frontend:** Next.js 15 (App Router), TypeScript, Tailwind CSS v4.0.
- **State Management:** Zustand (with Persistence).
- **Networking:** Axios with JWT interceptors.
- **Testing:** Jest + Supertest (Backend).
